=> There are some fields that are common in all the entities of the application.
In order to avoid repition of this again and again we can create a base class for all
these entities with the repeating fields as:

@MappedSuperclass // tells Spring that this is base class for all entity class extending this class
public class BaseEntity {
    @Column(updatable=false)
    private LocalDateTime createdAt; // column name will be 'created_at'
    @Column(updatable=false)
    private String createdBy;
    @Column(insertable=false)
    private LocalDateTime updatedAt;
    @Column(insertable=false)
    private String updatedBy;
}


=> What is "updatable = false": Tells JPA/Hibernate that this column should not be included in SQL UPDATE statements after the entity is initially persisted.

=> What is "insertable = false": This attribute in JPA/Hibernate tells the persistence provider not to include the column in SQL INSERT statements when a new entity is saved.
